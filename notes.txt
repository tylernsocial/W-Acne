What falls under inflammatory:

Papules (small red bumps)
Pustules (pus-filled bumps)
Nodules (deep, hard bumps)
Cysts (deep, fluid-filled)
Hormonal acne (same lesions, just triggered differently)
Nodulocystic acne (severe end of the spectrum)

Comedonal"These are clogged pores — not infected yet"
Inflammatory"This is active acne — bacteria is involved"
Pityrosporum"This looks like acne but is actually fungal"
Clear skin"Your skin is clear — you may be in maintenance phase"
Scarring"This is scarring/PIH — not active acne, needs different care"


Comedonal:

Include
Visible blackheads (open comedones)
Whiteheads (closed comedones)
Skin-colored bumps, no redness
Clogged pores with no inflammation

Exclude
Any redness or swelling present
Pus-filled lesions
Painful-looking deep bumps
Post-acne marks or pits

Edge case rule
One small red pimple present alongside comedones → label as inflammatory, not comedonal

Inflammatory (mild–severe)

Include
Red papules (solid red bumps)
Pustules (pus-filled with red base)
Nodules (deep, hard, painful-looking)
Cysts (deep, fluid-filled)
Hormonal pattern (jawline/chin)
Mixed comedonal + inflammatory

Exclude
No active red lesions present
Uniform tiny bumps with no redness (→ pityrosporum)
Only flat dark marks remaining
Only comedones, zero redness

Edge case rule
Active lesions + scarring in same image → label by the active acne, ignore the scarring

Pityrosporum folliculitis

Include
Uniform small itchy bumps
Located on chest, back, or shoulders
No blackheads or whiteheads present
Bumps all roughly same size
Confirmed fungal source (e.g. labeled dataset)

Exclude
Blackheads or whiteheads visible
Lesions of varying sizes
Located only on face (ambiguous)
Cannot confirm fungal origin

Edge case rule
If unsure whether it's pityrosporum or mild inflammatory, exclude the image entirely — this is your hardest class

Clear skin

Include
No active lesions of any kind
Smooth, even skin texture
Normal pores visible (not clogged)
Skin on face, cheek, forehead, chin

Exclude
Any active lesion present, even minor
Visible scarring or PIH marks
Redness or irritation visible
Image quality too low to confirm

Edge case rule
Skin with faded old marks but zero active lesions → label as clear skin, not scarring

Scarring

Include
Icepick, boxcar, or rolling scars
PIH — flat dark or red marks
Textural changes from past acne
No active lesions dominating image

Exclude
Active lesions present alongside marks
Marks so faint they read as clear skin
Non-acne scarring (cuts, burns)
Hyperpigmentation from other causes

Edge case rule
Active acne + scarring together → label as inflammatory, not scarring. Scarring label = scarring is the main feature

The baseline Logistic Regression model achieved a validation accuracy of 23.8%, which is only slightly above random guessing for a 5-class classification problem. This suggests that flattened pixel values are not enough to capture the visual patterns needed to distinguish acne categories.

The model performed best on inflammatory_acne and comedonal_open, but failed to correctly identify acne_excoriee, acne_scars, and comedonal_closed in the validation set. This is likely due to the small dataset size, class imbalance, and the loss of spatial information caused by flattening the images.

These results support the need for a CNN model, which can learn spatial features such as texture, local patterns, and lesion shape more effectively than a traditional flattened-pixel baseline.

Updated Baseline Model Interpretation

After increasing and balancing the dataset, the Logistic Regression baseline improved from 23.8% validation accuracy to 31.6%. This shows that improving dataset balance helped the model, but overall performance is still limited.

The model performed best on comedonal_open and inflammatory_acne, but struggled heavily with acne_excoriee and comedonal_closed. This is likely because Logistic Regression uses flattened pixel values, which removes spatial information and makes it difficult to learn subtle skin texture patterns.

These results suggest that a CNN is more appropriate for this task because it can learn local visual features such as lesion shape, texture, redness, and clustered patterns.


CNN Training Interpretation

The CNN training accuracy increased from around 21% to 54%, while the training loss steadily decreased. This shows that the model is learning patterns from the training set.

However, the validation accuracy fluctuated and did not consistently improve, while the validation loss increased after the early epochs. This suggests overfitting: the model is becoming better on the training images but is not generalizing well to unseen validation images.

Because the dataset is still relatively small, the validation accuracy is also unstable. Each validation image has a noticeable effect on the final percentage. The result shows that the CNN has more learning capacity than the baseline model, but additional regularization, data augmentation, and/or more training data are needed to improve generalization.


Second Baseline Model Interpretation

After removing the acne_excoriee class, the Logistic Regression baseline improved to 51.6% validation accuracy. Since random guessing for a 4-class problem would be 25%, this shows that the cleaned dataset contains learnable visual patterns.

The model performed best on inflammatory_acne and acne_scars, which are more visually distinct. It struggled most with comedonal_closed and comedonal_open, likely because these categories are visually similar and often overlap in real images.

This baseline provides a comparison point for the CNN model. Since Logistic Regression uses flattened pixel values and does not preserve spatial image structure, a CNN may improve performance by learning local texture, lesion shape, and visual patterns.

Second CNN Model Interpretation

The CNN reached around 71.8% training accuracy but only around 48.4% validation accuracy. The training loss steadily decreased, while the validation loss increased after the early epochs. This indicates overfitting: the model is learning the training images but is not generalizing well to unseen validation images.

Transfer Learning with MobileNetV2 Experiment

After testing a Logistic Regression baseline and several custom CNN models, the next experiment uses transfer learning. The custom CNN models struggled to generalize well, likely because the dataset is small and the classes are visually similar.

Transfer learning uses a pretrained model, MobileNetV2, as a feature extractor. The base model was trained on a large image dataset and has already learned general visual patterns such as edges, textures, and shapes. By freezing the base model and training only a new classification head, the model can adapt these learned features to the acne classification task without needing to learn all visual features from scratch.

Transfer Learning Interpretation

The MobileNetV2 transfer learning model performed significantly better than the custom CNN models. Validation accuracy reached around 67.7%, compared to 51.6% for the Logistic Regression baseline and below 50% for the custom CNN models.

Both training and validation accuracy increased together, and both training and validation loss decreased steadily. This suggests that the pretrained MobileNetV2 feature extractor is learning more generalizable image features than the CNN trained from scratch.

Because validation loss was still decreasing at the end of 30 epochs, the next experiment will train the transfer learning model for more epochs with early stopping.
